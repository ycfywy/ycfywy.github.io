ICLR 2025 论文阅读总结

标题 URL 摘要 status
Reasoning Elicitation in Language Models via Counterfactual Feedback link 提出反事实反馈机制，通过平衡事实与反事实问题的指标来增强 LLM 的因果推理能力。
ReGenesis: LLMs can Grow into Reasoning Generalists via Self-Improvement link 提出 ReGenesis 框架，通过从抽象到具体的自我合成推理路径，使模型在无监督下提升泛化推理能力。
Trust or Escalate: LLM Judges with Provable Guarantees for Human Agreement link 提出选择性评估框架，通过置信度估计确保 LLM 裁判与人类评价的一致性，不确定时升级到更强模型。
Do LLMs Recognize Your Preferences? Evaluating Personalized Preference Following in LLMs link 推出 PrefEval 基准测试，评估模型在长对话中推理、记忆和遵循用户个性化偏好的能力。
MAP: Multi-Human-Value Alignment Palette link 提出多人类价值对齐调色板（MAP），利用约束优化技术解决多样化且动态的人类价值对齐问题。
Turning Up the Heat: Min-p Sampling for Creative and Coherent LLM Outputs link 提出 min-p 采样方法，根据模型置信度动态调整阈值，在提升生成创造性的同时保持连贯性。
Backtracking Improves Generation Safety link 引入 [RESET] 令牌实现回溯技术，允许模型撤销并纠正已生成的有害文本，显著提升安全性。
From Exploration to Mastery: Enabling LLMs to Master Tools via Self-Driven Interactions link 提出 DRAFT 框架，通过 LLM 与工具的自我驱动交互（试错与反馈）来动态优化工具文档。
Mind the Gap: Examining the Self-Improvement Capabilities of Large Language Models link 研究 LLM 自我改进的数学机理，发现“生成-验证差距”与预训练算力规模之间存在单调比例关系。
Linear Representations of Political Perspective Emerge in Large Language Models link 揭示 LLM 内部存在政治立场的线性表征，并展示了如何通过干预特定注意力头来控制输出倾向。
Self-Improvement in Language Models: The Sharpening Mechanism link 提出“锐化”理论视角，利用模型的验证能力在后训练阶段优化生成分布，降低推理成本。
A Decade’s Battle on Dataset Bias: Are We There Yet? link 重新审视数据集偏差，发现现代模型仍能轻易识别图像所属数据集，暗示数据偏差问题依然严峻。
Limits to scalable evaluation at the frontier: LLM as judge won’t beat twice the data link 论证了“LLM 作为裁判”的理论极限，指出当裁判能力不足时，无法通过算法手段显著减少对人工标签的需求。
